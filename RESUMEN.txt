=========================================================================

ARQUITECTURA ORIGINAL: MONOLÍTICA
---------------------------------
El proyecto original consistía en una aplicación React simple de carrito de 
compras implementada completamente en un solo componente (App.jsx) con 
aproximadamente 154 líneas de código. La aplicación seguía un patrón 
monolítico donde toda la lógica estaba mezclada sin separación de 
responsabilidades.

ESTRUCTURA DE ARCHIVOS ORIGINAL:
src/
├── App.jsx          (154 líneas - TODO EN UN ARCHIVO)
└── main.jsx         (Punto de entrada básico)

FUNCIONALIDADES IMPLEMENTADAS:
- Lista de productos (Mouse $20, Teclado $35, Monitor $150)
- Carrito de compras con cantidad modificable
- Descuentos para usuarios premium (5%)
- Sistema de cupones (PROMO10: -10%, FIJO20: -$20)
- Cálculo de impuestos por región (CR: 13%, US-CA: 7.25%, US-TX: 6.25%, OTRA: 10%)
- Cálculo de total final con todos los descuentos e impuestos

PROBLEMAS IDENTIFICADOS EN LA VERSIÓN ORIGINAL:
----------------------------------------------
1. ACOPLAMIENTO ALTO: Toda la lógica en un solo componente
2. MEZCLA DE RESPONSABILIDADES: UI, lógica de negocio y cálculos juntos
3. CÓDIGO DUPLICADO: Lógica de formateo repetida
4. DIFICULTAD PARA TESTING: Efectos secundarios mezclados con lógica
5. MANTENIMIENTO COMPLEJO: Cambios requieren modificar todo el archivo
6. ESCALABILIDAD LIMITADA: Difícil agregar nuevas funcionalidades
7. LOGGING INCONSISTENTE: console.log dispersos sin estructura
8. FALTA DE REUTILIZACIÓN: Componentes no modulares
9. VIOLACIÓN DE PRINCIPIOS SOLID: Múltiples responsabilidades por clase
10. AUSENCIA DE PATRONES: No hay separación de capas ni patrones aplicados

FUNCIÓN: Cálculo de subtotal

ORIGINAL (en App.jsx):
```javascript
// Mezclado con otros cálculos en recalc()
let subtotal = 0;
for (const item of cartArg) {
  subtotal += (item.price || 0) * (item.qty || 0);
}
```

ACTUAL (en domain/checkout.js):
```javascript
export function computeSubtotal(cartItems) {
  if (!Array.isArray(cartItems)) return 0;
  let subtotal = 0;
  for (const item of cartItems) {
    const price = Number(item?.price) || 0;
    const qty = Number(item?.qty) || 0;
    if (!isValidMoneyValue(price) || qty < 0) continue;
    subtotal += price * qty;
  }
  return roundMoney(subtotal);
}
```
